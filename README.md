# Простой Калькулятор на Tkinter

Реализация приложения "Калькулятор" на Python с использованием библиотеки `tkinter`.
Приложение поддерживает базовые арифметические операции, вычисление квадратного корня.

## Функционал

*   **Базовые операции:** Сложение, вычитание, умножение, деление.
*   **Дополнительно:** Взятие квадратного корня (√).
*   **Валидация:**
    *   Обработка деления на ноль.
    *   Обработка попытки взять корень из отрицательного числа.
    *   Защита от ввода недопустимых символов.
*   **Интерфейс:**
    *   Стилизованные круглые кнопки (`RoundedButton`).
    *   Ввод как с клавиатуры, так и через интерфейс.
    *   Поддержка клавиш `Enter` (равно) и `Backspace` (удаление).

## Технологии

*   **Python 3.x**
*   **Tkinter** (стандартная библиотека для GUI)
*   **Math** (математические функции)

## Установка и Запуск


### 1. Клонирование репозитория

```bash
git clone https://github.com/Ficus-aurea/Calculator.git
```

```bash
pip install -r requirements.txt
```

```bash
cd Calculator
```


### 2. Запуск приложения

```bash
python calculator.py
```
